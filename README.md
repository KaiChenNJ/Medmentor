# MedMentor

Official implementation of **MedMentor** — a Teacher–Student multimodal multi-agent framework for medical imaging and clinical reasoning.

## Quick Start

1. Install deps

   ```bash
   pip install -r requirements.txt
   ```
2. Configure your API key in `config.yaml`.
3. Launch the app

   ```bash
   python main.py
   ```
4. Open the Flask UI
